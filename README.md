Generate and improve developer resumes with AI.

## Install

```bash
npm install -g ai-resume
```

## Usage

```bash
npx ai-resume "senior fullstack engineer, 5 years React, Node.js, AWS"
npx ai-resume "junior frontend dev, 1 year Vue.js" -o my-resume.md
```
